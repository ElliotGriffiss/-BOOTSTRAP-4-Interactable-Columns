Short Description

I made this project to test out a clean way of creating responsive gutters in Bootstrap 4.

The main challenge of this project was creating the gutters without the .column-cover overflowing past the .column-image and overlapping the background color.

This project is set up to work with images rather than colours, if you replace the inline css style="background-color: #5aa7de;" with style="background-image: */your image*" it will still work as intended.
